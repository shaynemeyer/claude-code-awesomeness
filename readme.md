# Claude Code Awesomeness

A comprehensive collection of guides and references for mastering Claude Code - Anthropic's official CLI tool for AI-assisted software engineering.

## What's Inside

This repository contains in-depth documentation covering Claude Code's architecture, features, and best practices:

### Core Guides

- **[Mastery Guide](docs/claude-code-mastery-guide.md)** - Deep dive into architecture, internals, and the agentic loop. Perfect for understanding how Claude Code works under the hood.

- **[Summary with Diagrams](docs/claude-code-summary-with-diagrams.md)** - Visual overview using Mermaid diagrams. Great starting point for visual learners covering all core concepts.

- **[Hooks Guide](docs/claude-code-hooks-guide.md)** - Complete reference for the hooks system (PreToolUse, PostToolUse, UserPromptSubmit, SessionStart) for deterministic automation.

- **[Skills Guide](docs/claude-code-skills-guide.md)** - Creating and using skills for auto-applied domain expertise and coding standards.

- **[Plugins Guide](docs/claude-code-plugins-guide.md)** - Plugin system for packaging and distributing Claude Code extensions.

## Key Topics Covered

- Three-layer architecture (Core, Delegation, Extension)
- Model selection strategies (Haiku, Sonnet, Opus, Opusplan)
- Configuration hierarchy and settings management
- Permission system with allow/deny rules
- MCP (Model Context Protocol) for 300+ integrations
- Context management and cost optimization
- Subagents for parallel execution
- Enterprise deployment patterns
